# mininote
mininote 는 가볍고 빠른 일정관리 툴입니다
에버노트는 무겁고 무료버젼에는 기능상 제약이 많습니다. mininote 는 가장 단순하고 핵심적인 몇가지 기능들만을 제공하되 쉽고 빠르고 어느 기기에서나 사용이 가능합니다


#### 기능설명
* 단축키
  * Alt(or meta) + W : 글쓰기
  * Alt(or meta) + L : 목록보기
  * Alt(or meta) + Enter : 저장/조회
* 예약어 autoReplace
  * mo + "띄어쓰기" :  MO todolist
  * tu + "띄어쓰기" :  TU todolist
  * we + "띄어쓰기" :  WE todolist
  * th + "띄어쓰기" :  TH todolist
  * fr + "띄어쓰기" :  FR todolist
  * !! + "띄어쓰기" : 체크박스 하나 추가
  * @@ + "띄어쓰기" : 들여쓰기
